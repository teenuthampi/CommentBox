# Post Commenting 

## Getting Started

### Requirements

1. Ruby 2.4.1
2. MySQL 
3. Rails 5

### Setup


```
https://github.com/teenuthampi/CommentBox.git
cd CommentBox
bundle
rails db:drop db:create db:migrate 
```

### Testing

```
bundle exec rspec
```


### Running Application

```
bundle exec rails s
```

The app should be available in development at http://localhost:3000.
