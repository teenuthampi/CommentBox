# Post Commenting 

## Getting Started

### Requirements

1. Ruby 2.4.1
2. MySQL 
3. Rails 5

### Setup


```
https://github.com/teenuthampi/CommentBox.git
cd CommentBox
bundle
```

### Testing

```
bundle exec rspec
```

### Integration Testing
```
bundle exec cucumber
```

### Running Application

```
bundle exec rails s
```

The app should be available in development at http://localhost:3000.
